About

This is the central aggregation server for UnifiedEdu, a privacy-preserving federated learning framework for automated educational quiz generation. Collaborating institutions train a quiz generation model locally on their own data and submit only a compact GNN parameter vector to this server. The server aggregates updates via weighted FedAvg and returns an improved global model — your raw data and full model weights never leave your machine. The framework is fully architecture-agnostic: BERT, LLaMA, T5, Qwen, or any custom PyTorch model is supported via a shared graph-based representation space.

API Endpoints

Available Routes
GET / This page — framework overview and usage guide
GET /health Server health check, current round, and mode status
GET /get_global_parameters Download the latest global GNN parameters before local training
POST /aggregate Submit your local GNN update — receive the aggregated global model in return
GET /round_status Check how many clients have submitted in the current round

POST /aggregate — Payload Schema

json
{
  "client_id":   "your_institution_name",  // string — unique ID given to you
  "parameters":  [0.012, -0.034,...],    // float array — flat GNN param vector
  "num_samples": 1200,                    // int — your local dataset size
  "round":       3                        // int — current round number
}

Quick Start

bash · step 1 — install
git clone https://github.com/furkanpala/UnifiedEdu
cd UnifiedEdu
pip install -r requirements.txt
python · step 2 — run federated rounds
from unified_client import UnifiedClient
from transformers import AutoModel

# Load your local model — stays on your machine
model = AutoModel.from_pretrained("bert-base-uncased")

# Wrap it with UnifiedClient
client = UnifiedClient(
    client_id   = "your_institution",
    server_url  = "https://unifiededu.onrender.com",
    model       = model,
    data_path   = "your_data.jsonl",
    num_samples = 1200,
)

# Run federated rounds — one line per round
for round_idx in range(10):
    result = client.run_round()
    print(f"Round {result['round']} | Loss: {result['avg_loss']:.4f}")
